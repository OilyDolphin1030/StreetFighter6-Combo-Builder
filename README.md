SF6 Combo Icon Builder
Street Fighter 6 のコンボを、アイコン付きで作成・保存・共有できるWebツールです。
公開ページ
https://oilydolphin1030.github.io/StreetFighter6-Combo-Builder/
主な機能
矢印・攻撃ボタンのアイコン入力
テキスト貼り付けからの画像化
キャラ名・コンボ名付きで保存
保存済みコンボの確認
コンボの並び替え
不要なコンボの削除
並びテキストのコピー
画像としてコピー
共有ファイルの書き出し
共有ファイルの読み込み
使い方
キャラ名を入力
コンボ名を入力
矢印・攻撃ボタンを押してコンボを作成
「コンボ保存」で保存
必要に応じて、画像コピーまたは共有ファイルを書き出し
こんな用途に
トレモ中のコンボメモ
キャラ別コンボリスト作成
サブキャラのコンボ管理
友人やDiscordでのコンボ共有
noteやSNS向けのコンボ画像作成
